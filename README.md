# 🤖 Brixel WebEditor (Blockcoding WebIDE)

**"The ultimate solution for physical computing education and practice."**

Brixel WebEditor is a powerful web-based block coding platform that allows users to code and upload to various microcontroller boards, including **Arduino**, **ESP32**, and **Raspberry Pi Pico**, from a single unified interface.

---

## ✨ Key Features (English)

### 1. 🌐 Universal Multi-Board Support
*   **Perfect Compatibility**: Seamlessly upload to Arduino Uno, ESP32, and Raspberry Pi Pico (RP2040) without manual configuration changes.
*   **Write Once, Run Anywhere**: The code logic remains consistent regardless of the target board.

### 2. 🇰🇷 Korean-Friendly Smart Romanization
*   **Han-geul Variable Names**: Supports Korean variable names in blocks.
*   **Auto-Romanization**: Automatically converts Korean variables to Romanized strings during compilation to solve C++ compiler limitations.

### 3. 📷 ESP32-CAM & AIoT Integration
*   **Dedicated CAM Blocks**: Easy implementation of camera initialization, web streaming, and image processing.
*   **AIoT Features**: Ready for data-driven AIoT projects.

### 4. 🧱 Massive Hardware Library
*   **100+ I/O Devices**: Pre-built blocks for over 100 sensors, motors, and displays.

### 5. 📝 Assembly Recording System
*   **Creative Journey Tracker**: Records the step-by-step block assembly process for review and educational assessment.

### 6. 🌍 Global Multi-language Support
*   **Bilingual Interface**: Seamlessly switch between English and Korean.

브릭셀 웹에디터는 아두이노(Arduino), ESP32, 라즈베리 파이 피코(Raspberry Pi Pico) 등 다양한 보드를 하나의 인터페이스에서 코딩하고 업로드할 수 있는 웹 기반 블록 코딩 플랫폼입니다.

---

## ✨ 주요 기능 (한국어)

### 1. 🌐 완벽한 멀티 보드 지원
*   **통합 업로드**: 아두이노 에스더(Uno), ESP32, 라즈베리 파이 피코(RP2040) 보드를 별도 설정 없이 즉시 지원합니다.
*   **코드 호환성**: 하나의 블록 코드로 모든 보드에서 동일하게 작동하는 진정한 '원소스 멀티유즈'를 실현했습니다.

### 2. 🇰🇷 한글 친화적 스마트 로마자 변환
*   **한글 변수명 지원**: 블록 프로그래밍 시 '온도', '습도' 등 한글 변수명을 자유롭게 사용할 수 있습니다.
*   **자동 변환 엔진**: 컴파일 시 한글을 로마자(예: `ondo`)로 자동 변환하여 C++ 컴파일러의 한글 미지원 문제를 원천적으로 해결했습니다.

### 3. 📷 ESP32-CAM & AIoT 완벽 지원
*   **전용 카메라 블록**: 복잡한 카메라 설정, 웹 스트리밍, 영상 처리 코드를 몇 개의 블록으로 간편하게 구현합니다.
*   **AIoT 확장성**: 센서 데이터 수집 및 분석을 통한 AIoT 프로젝트에 최적화되어 있습니다.

### 4. 🧱 방대한 하드웨어 라이브러리
*   **100종 이상의 장치 지원**: 100여 종 이상의 센서, 모터, 디스플레이 전용 블록을 기본 내장하여 별도 라이브러리 설치가 필요 없습니다.

### 5. 📝 블록 조립 기록 기능 (Assembly Record)
*   **창의적 과정 기록**: 학습자가 블록을 조립하는 모든 과정을 기록하여, 문제 해결 능력 평가 및 복습에 활용할 수 있습니다.

### 6. 🌍 글로벌 다국어 지원
*   **다국어 환경**: 한국어와 영어를 완벽하게 지원하며, 설정에서 자유롭게 전환이 가능합니다.
