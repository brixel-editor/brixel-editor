 🛠️ 기술 스택

  프론트엔드:
  - Blockly 12.2.0 (블록 프로그래밍)
  - Monaco Editor 0.34.0 (VS Code 기반 텍스트 에디터)
  - Vanilla JavaScript ES6+ (모듈화 구조)

  백엔드 (PC 에이전트):
  - Node.js + Electron
  - Express (HTTP REST API)
  - WebSocket (실시간 상태 통신)
  - serialport (아두이노/ESP32 시리얼 통신)
  - Arduino CLI (크로스 플랫폼 컴파일/업로드)

  ⚙️ 주요 기능

  1. 블록 프로그래밍: Blockly 기반 비주얼 코딩, 직관적인 UI
  2. 스마트 코드 생성: 
     - 블록 → C++ 코드 자동 변환
     - **한글 변수명 자동 로마자 변환 지원 (NEW)**
  3. 강력한 하드웨어 호환성: 
     - Arduino, ESP32, Raspberry Pi Pico 등 다중 보드 지원
     - 100종 이상의 입출력 장치/센서 블록 기본 내장
  4. ESP32-CAM 지원: 카메라 설정, 웹 스트리밍, 영상 처리 블록 내장
  5. 컴파일/업로드: 로컬 에이전트를 통한 원클릭 펌웨어 플래싱
  6. 사용자 편의:
     - 사라진 블록 자동 감지 및 알림 (Red Block Alert)
     - 블록 조립 과정 자동 기록 (Assembly Record)
  7. 다국어 지원: 글로벌 교육 환경을 위한 다국어 시스템

  🔄 통신 구조

  웹 브라우저 (index.html)
      ↕ HTTP (명령) / WebSocket (상태/로그)
  PC 에이전트 (Electron/Node.js)
      ↕ Serial/USB (COM 포트)
  마이크로컨트롤러 (Arduino, ESP32, Pico)
